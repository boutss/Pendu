== Présentation

*Le Pendu* est une application web de jeu du pendu complète, jouable depuis n'importe quel navigateur moderne.
Le joueur doit deviner un mot lettre par lettre avant d'épuiser ses tentatives, sous peine de voir le pendu se dessiner progressivement.

== Fonctionnalités

* 🎨 *Dessin SVG progressif* du pendu en 6 étapes
* 🗂️ *5 catégories* de mots : Animaux, Pays, Sports, Films, Aliments
* ⚡ *3 niveaux de difficulté* — Facile, Moyen, Difficile
* 💡 *Indices* disponibles en mode Facile et Moyen
* ⌨️ *Clavier virtuel* cliquable + saisie au clavier physique
* 🏆 *Système de score* avec bonus de série et de difficulté
* 💾 *Sauvegarde locale* du top 5 des meilleurs scores (localStorage)
* 📱 *Responsive* — fonctionne sur mobile et desktop

== Niveaux de difficulté

[cols="1,1,1", options="header"]
|===
| Niveau | Vies | Indice affiché
| Facile | 8 | ✅ Oui
| Moyen | 6 | ✅ Oui
| Difficile | 5 | ❌ Non
|===

== Calcul des points

[source]
----
Points de base  = vies restantes × 10
Multiplicateur  = ×1.5 (Moyen) ou ×2.5 (Difficile)
Bonus de série  = série × 5 points (à partir du 2e mot consécutif)
----

== Structure du projet

[source]
----
Pendu/
├── index.html   ← Structure des 3 écrans (accueil, jeu, résultat)

----

== Lancer le projet en local

Cloner le dépôt :

[source,bash]
----
git clone https://github.com/boutss/Pendu.git
cd Pendu
----

Ouvrir `index.html` dans un navigateur — aucune dépendance, aucun serveur requis.
